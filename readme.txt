This plugin brings Related post slider to the WordPress blog post.

== Description ==

You can easily show related posts thumbnail,content, title after the post. You remember it that it will be show by Category.First, you will active thsi plugin then it will working. This fast to use and configure intuitive and the best thing is free.

- Add Category of your post

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "related post with slider"
3. **Install** the "related post with slider" plugin
4. **Activate** "related post with slider" from your Plugins page
5. **Dashboard** go to post & give your category then it will be show on your post
6. **Dashboard** after installing you can see "Dashboard" a new menu "Related Post".

**Manually Install**
1. **Download** Link :: <a href="https://wordpress.org/plugins/related-posts-with-slider/">Download Link</a>
2. **Copy & Past** go >> plugin directory >> past >> active
